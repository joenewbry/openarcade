# Resource Chain

- Regolith
- Ilmenite

- Basalt

## Key Conversions
- Ilmenite -> aluminum feedstock + oxygen precursors
- Ice -> oxygen + hydrogen
- Basalt -> structural ceramics

## Program Outputs
- Rail segments
- Capacitor cores
- Guidance modules

These outputs feed directly into Mass Driver Program stages.
